# Shravan Parthe — Portfolio

A professional developer portfolio built with Streamlit, featuring a dark-themed GitHub-inspired design.

## Features

- Dark professional theme with custom CSS
- Responsive layout (desktop & mobile)
- Sections: Hero, Skills, Projects, Experience, Education, Contact
- Color-coded skill categories (Languages, Frameworks, Tools, AI/ML)
- Project cards with tech stack chips and GitHub links
- Timeline-style experience and education cards

## Deploy on Streamlit Cloud

1. Push this repository to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your GitHub account
4. Select this repository → `app.py`
5. Click **Deploy**

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Tech Stack

- Python 3.8+
- Streamlit
- Custom HTML/CSS (Cabinet Grotesk + General Sans fonts)
